## Usage

```
# サーバーを起動する
$ ./dc.sh start

# サーバーを停止する
$ ./dc.sh stop
```

### AWStatusコンソールを表示する
http://localhost:8090/

### ログファイルをAWStatsに取り込みする

```
# ログファイルのインポート
$ ./import.sh
```

## Install

```
$ ./dc.sh init

$ ./dc.sh start
```
